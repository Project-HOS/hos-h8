＜はじめに＞

  この度は Hyper Oparating System V3 for Hitachi H8 (以下 HOS-H8)を
ダウンロードいただき誠に有り難うございます。

  HOS-H8 は日立 H8/300H 用のμITRON3.0仕様のレベルＳまでをサポートした
リアルタイムオペレーティングシステムとしてスタートしました。



＜ソースについて＞

  ソースは 秋月電子から販売されるＣコンパイラやアセンブラに対応した
形式となっています。恐らく日立純正コンパイラでもそのままコンパイル可能だと
おもいます（ライブラリアンがあるのでしたらソースをモジュール毎に分離
したほうが良いでしょう）。
  なおタブ幅はＣでは４、アセンブラでは８に設定して書いております。



＜コンフィギュレーターについて＞

  コンフィギュレータについては本当はソースは公開したくなかったんですが
（あまりにいい加減で恥ずかしすぎる）、そのままでは実用に耐えない可能性が
高いので一応つけました。C++の機能は使っていないのでほとんどのコンパイラで
コンパイル可能かとおもいます。もっともたいしたことはしていないので不満の
ある方は１から書き直した方が早いかと思います(^^;

＜HOS-H8 の歩み＞

  Hyper Operating System は、1998年 当時学生だった、Ryuz によって
Z80用のμITRON仕様のものを HOS-80 としてリリースしたのが始まりと
なります。その後、H8用に改造したものがリリースされ、現在は
Project HOS に全資産が移管され、インターネット上の有志によって、
開発及び保守作業が進められております。
  Project HOS はHOSに興味のある方なら誰でも参加できますし、直接参加
ではありませんが、バグレポやご意見をお寄せいただくことで間接的に
開発にご協力頂いている多くの方により運営が成立しております。
  是非とも http://sourceforge.jp/projects/hos にお立ちより頂ければ
幸いです。

  なお、Project HOS の擁護の為に Ryuz より一言加えておきますと、
現時点での問題点の多くはオープンソース開発に移行後の日が浅く、オリジナル
ソースがまだ大部分残っていることに起因しております。
  今後オープンソースの世界において、Project HOS のメンバーと、皆様の
ご協力の中で新たに育てていただけますことを切に祈っております。
